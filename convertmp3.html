Conversor Vídeo → MP3 (local)

Escolha vídeos do seu computador, converta para MP3 no navegador (ffmpeg.wasm). Você pode salvar os arquivos convertidos e/ou os vídeos originais em pastas locais (File System Access API) ou baixar manualmente se o navegador não suportar.

Pronto.
Nenhuma tarefa iniciada.
Observações: Esta aplicação roda totalmente no navegador usando ffmpeg.wasm. A criação/seleção de pastas locais funciona em navegadores baseados em Chromium (Chrome, Edge e derivados) que suportam a File System Access API. Se o navegador não suportar, o arquivo convertido será baixado via link.
